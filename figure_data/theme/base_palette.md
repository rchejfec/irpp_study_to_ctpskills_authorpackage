# Base Brand Palette

This is the raw, constrained palette extracted from the organization's style guide. The hex values below have been checked against the official RGB values, confirming that the two Infogram typos (Dark Blue and Sage Green) were indeed correctly fixed in the original `theme.css`.

Instead of opinionated or project-specific names (like `irpp-navy` or `viable`), these are grouped by descriptive color families. Each family provides a "Base" (darker/saturated) and a "Light" (tinted/pastel) variant.

## 1. Blues & Teals
*   **Dark Blue**

    *   Hex: `#004667` *(Note: Fixed from Infogram's `004677`)*
*   **Light Blue**
    *   RGB: 184, 196, 211
    *   Hex: `#B8C4D3`
*   **Teal**
    *   RGB: 68, 163, 164
